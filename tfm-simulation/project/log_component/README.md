# log_component

Este componente simula un proceso de compilación con muchos logs de diferentes niveles (INFO, DEBUG, WARNING, ERROR).

## Archivos
- `build.bat`: Script de compilación que imprime muchos logs.
- `log_main.cpp`: Código fuente principal que genera logs durante la ejecución.
- `startup.sh`: Script de arranque para ejecutar el componente.

## Uso

### Compilación (Windows)

```
cd log_component
build.bat
```

### Ejecución (Windows)

```
log_component.exe
```

### Ejecución (Linux, usando WSL o similar)

```
./startup.sh
```
